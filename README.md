🎓 Student AI Performance Dashboard
📌 Quick Summary
An interactive Data Science product that analyzes student test scores and uses a Linear Regression model to predict Math results based on Reading and Writing performance.
https://studentdashboard-sqrock-week2-task1-sysvhdonzyjr69zpksy2ph.streamlit.app/

1. Live Dashboard

🛠️ Tech Stack
Analytics: Python, Pandas

Visuals: Plotly, Seaborn

AI Model: Scikit-Learn (Linear Regression)

Interface: Streamlit

🚀 Key Features
EDA: Automated cleaning and handling of missing data.

Visuals: Distribution histograms and correlation heatmaps.

AI Predictor: Real-time Math score prediction with a confidence gauge.

Performance: Model achieved a 0.68 R2 Score.
